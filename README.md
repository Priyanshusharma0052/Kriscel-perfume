# Kriscel-perfume (NOIR)

A premium perfume showcase and shopping experience built with React + Vite.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Collection carousel with detailed perfume cards
- Finder: select Top/Heart/Base notes to get a single best-match recommendation
- Occasional Edit: curated recommendations by occasion
- Perfume details modal + availability request flow
